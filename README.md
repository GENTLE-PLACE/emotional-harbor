# 🌊 Эмоциональная Гавань

Твоё нежное пространство для внутренней жизни — честной, тёплой и без давления «быть продуктивной» или «всегда позитивной».

Это место, где можно просто быть собой: замечать оттенки эмоций, собирать маленькие радости, опираться на свои ресурсы и медленно формировать привычку заботы о себе.

### 🎯 Что здесь есть

- **onboarding-home.html** — главное руководство: письмо от автора, знакомство с проводником Есенией, карта Гавани и первый шаг за 15 минут.
- **onboarding-diary.html** — подробное руководство, как вести Дневник настроения (настроение, энергия, эмоции, три радости дня и многое другое).
- Шрифты, стили и другие файлы для корректной работы страниц.

### 💡 Для чего это пространство

Эмоциональная Гавань помогает развивать **эмоциональную гранулярность** — умение точно называть свои чувства.  
Основано на исследованиях:
- Martin Seligman (фиксация радостей)
- Lisa Feldman Barrett (называние эмоций)
- James Pennebaker (экспрессивное письмо)
- и других работах по позитивной психологии.

Здесь нет «правильных» ответов и жёстких норм. Только честность с собой и маленькие шаги по 3 минуты в день.

Если хочешь подключить стили, шрифты или другие ресурсы из этого репозитория в свои проекты — используй **Raw**-ссылки:

1. Открой нужный файл.
2. Нажми кнопку **Raw**.
3. Скопируй ссылку из адресной строки.
